# Guía de Usuario - EAFITos

Bienvenido a la shell oficial de sistemas operativos. EAFITos te permite interactuar con tu sistema de archivos de forma simplificada.

## Uso de Comandos Especiales

### Búsqueda de Texto (`buscar`)
Busca una palabra dentro de un archivo sin importar mayúsculas/minúsculas.
**Sintaxis**: `buscar <palabra> <archivo>`
**Ejemplo**: `buscar hola notas.txt`

### Gestión de Archivos
- **Crear**: `crear <nombre>` (Crea un archivo vacío).
- **Eliminar**: `eliminar <nombre>` (Borra permanentemente el archivo).
- **Renombrar**: `renombrar <viejo> <nuevo>` (Cambia el nombre o mueve el archivo).

## Tips de Navegación
- Usa el comando `limpiar` si la pantalla está muy saturada.
- El comando `calc` requiere espacios entre los números y el operador: `calc 5 + 10`.
- Para salir de la shell, escribe `salir` o usa el atajo `Ctrl + D`.
